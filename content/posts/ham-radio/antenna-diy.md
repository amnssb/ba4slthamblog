## 项目背景

HF 通联需要一副可靠的天线。 commercially available 的天线价格不菲，而且不一定适合自己的使用环境。于是决定自己动手制作一副半波偶极天线。

## 设计参数

| 参数 | 数值 |
|------|------|
| 频段 | 7.0-7.3 MHz (40m) |
| 类型 | 半波偶极 (Dipole) |
| 总长度 | 约 20.3 米 |
| 单臂长度 | 约 10.15 米 |
| 线材 | 2.5mm² 多股铜线 |

## 计算公式

半波偶极天线的长度计算公式：

```
L(m) = 142.5 / f(MHz)
```

对于 7.1 MHz 中心频率：

```
L = 142.5 / 7.1 ≈ 20.07 米
```

考虑到末端效应，实际制作时增加了 3%。

## 制作步骤

### 1. 准备材料

- 2.5mm² 多股铜线 21 米
- 1:1 巴伦 (Balun)
- 绝缘子 2 个
- 尼龙绳若干
- 50Ω 同轴电缆 15 米

### 2. 裁剪线材

按照计算长度裁剪两段 10.15 米的线材。尽量做到长度一致，误差控制在 1cm 以内。

### 3. 连接巴伦

将两段线材的一端分别连接到巴伦的两个输出端。使用焊锡焊接，确保连接牢固。

### 4. 安装绝缘子

在另一端的线材末端安装绝缘子，用于固定天线。

### 5. 架设天线

我选择"倒 V" 架设方式：

- 中心点高度：约 8 米
- 两端高度：约 2 米
- 夹角：约 120°

## VSWR 测量

使用天线分析仪测量结果：

| 频率 | VSWR |
|------|------|
| 7.000 MHz | 1:1.8 |
| 7.100 MHz | 1:1.2 |
| 7.200 MHz | 1:1.4 |
| 7.300 MHz | 1:1.9 |

在 7.0-7.2 MHz 范围内 VSWR < 1:1.5，效果非常好！

## 使用效果

- 接收灵敏度明显提升
- 发射功率可以被有效辐射
- 成功完成多次 1000km+ 通联

## 总结

自制天线的关键是：**精确的长度 + 可靠的连接 + 合理的架设**。

总成本不到 100 元，性价比极高。推荐给各位想尝试 DIY 的火腿朋友！
